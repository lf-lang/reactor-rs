    (T0) => { ... };
    (T0, $microstep:expr) => { ... };

}
Expand description

Convenient macro to create a tag. This is just a shorthand for using the constructor together with the syntax of delay.

use reactor_rt::{tag, delay};

tag!(T0 + 20 ms);
tag!(T0 + 60 ms);
tag!(T0); // the origin tag
// with a microstep:
tag!(T0, 1);
tag!(T0 + 3 sec, 1);
